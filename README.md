

# 📉 Product Performance & Inventory Analytics

## 🚀 Project Summary

The **Product Performance & Inventory Analytics** is an end-to-end business intelligence project developed to analyze declining sales performance and generate actionable insights for revenue improvement.

This project focuses on transforming raw retail transaction data into meaningful business insights using interactive dashboards. It helps stakeholders understand **why sales are declining, where performance gaps exist, and what strategic actions can be taken** to improve business outcomes.

---

## 🎯 Problem Statement

A retail organization observed a consistent **decline in overall sales revenue and profitability** across multiple regions and product categories. However, decision-makers lacked a centralized analytics solution to:

* Track performance trends
* Identify underperforming regions
* Analyze customer purchase behavior
* Evaluate product demand fluctuations
* Monitor profitability and operational efficiency

This project was developed to solve these challenges through data-driven analytics.

---

## 📂 Dataset Information

The dataset contains retail transactional records including:

* Order ID
* Order Date
* Region
* Customer ID
* Product Category
* Product Name
* Sales Amount
* Quantity
* Cost
* Profit

The dataset was cleaned and transformed to ensure data consistency and analytical accuracy.

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard design and visualization
* **Power Query** – Data cleaning and transformation
* **DAX** – KPI creation and business logic
* **SQL** – Data extraction and aggregation
* **Excel / CSV** – Data source

---

## 🔄 Data Analysis Workflow

### 1️⃣ Data Cleaning

* Removed null and duplicate records
* Standardized category and region naming
* Created calculated columns for profit and margin

### 2️⃣ Data Modeling

* Built star schema with fact and dimension tables
* Established relationships between sales, customers, products, and regions

### 3️⃣ KPI Development

* Created measures for revenue, profit margin, growth %, and order quantity
* Designed dynamic KPIs using DAX

### 4️⃣ Dashboard Development

* Built interactive visuals using slicers and filters
* Implemented drill-down capability for detailed analysis
* Designed executive summary view

## 🔍 Key Business Insights

* Certain regions contributed significantly to overall revenue decline
* Specific product categories showed consistent low demand
* Repeat customer purchases reduced during selected periods
* Profit margins were impacted due to high operational costs
* Pricing and inventory optimization opportunities were identified

## 💡 Business Recommendations

* Focus marketing efforts on underperforming regions
* Optimize pricing strategy for low margin products
* Improve inventory planning based on demand trends
* Implement customer retention programs
* Monitor KPIs regularly using dashboard
